## Warning

This project is locked to **Godot 4.2**. Do NOT use APIs, features, or patterns introduced in:
- Godot 4.3+ (GDExtension stabilization changes, C# 12)
- Godot 4.5+ (@abstract, variadic arguments, Shader Baker, SMAA 1x)
- Godot 4.6+ (SkeletonModifier3D IK, D3D12 default, AgX tonemapper, glow rework)

Always consult this directory before suggesting Godot API usage. When in doubt, use the 4.2 official documentation: https://docs.godotengine.org/en/4.2/

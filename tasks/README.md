# DarkAges Tasks

This directory contains executable implementation tasks for the development roadmap.

## Task Structure

Each task file contains:
- **Priority**: P0 (MVP Critical), P1 (Core), P2 (Gameplay), P3 (Polish)
- **PRD Reference**: Link to implementation specification
- **Files to create**: Exact paths
- **Integration points**: Where to wire the system
- **Acceptance criteria**: Verified testable checkpoints

## Task Execution

1. **Select a task** from the list below
2. **Read the task file** for exact requirements
3. **Check the implementation PRD** in `../prd/` for code specs
4. **Implement to spec**
5. **Write tests** covering acceptance criteria
6. **Update task status** when complete
7. **Update AGENTS.md** when feature is done

# Task T1.1: Player Persistence Implementation

## Priority
**P0 - MVP Critical** (All progress currently lost on disconnect)

## PRD Reference
`prd/prd-player-persistence.md`

## Description
Implement player data persistence so character progress (level, inventory, achievements, faction standing) survives server restarts and disconnects.

## Class Signature

### PlayerProfile.hpp
```cpp
namespace DarkAges::memory {

struct PlayerProfile {
  uint64_t account_id;
  uint64_t character_id;
  std::string name;
  uint8_t level;
  uint64_t experience;
  uint64_t currency;
  
  // Inventory (slot -> item)
  std::unordered_map<uint8_t, InventoryItem> inventory;
  
  // Equipment (slot -> item)
  std::unordered_map<uint8_t, InventoryItem> equipment;
  
  // Quest progress
  std::unordered_map<uint32_t, QuestProgress> quests;
  
  // Achievements
  std::vector<uint32_t> achievements;
  
  // Faction standing
  std::unordered_map<uint32_t, int32_t> faction_standing;
  
  std::chrono::system_clock::time_point last_login;
};

}
```

### PlayerProfileStore.hpp
```cpp
namespace DarkAges::memory {

class PlayerProfileStore {
public:
  // Create new character
  uint64_t CreateCharacter(uint64_t account_id, const std::string& name, uint8_t class_id);
  
  // Load character on login
  std::optional<PlayerProfile> LoadCharacter(uint64_t character_id);
  
  // Save character (debounced)
  void SaveCharacter(const PlayerProfile& profile);
  
  // Auto-save callback
  void OnTick();
  
  // Get all characters for account
  std::vector<CharacterInfo> GetCharacters(uint64_t account_id);
  
  // Delete character
  bool DeleteCharacter(uint64_t character_id);
};

}
```
